## 🛠️ How it Works

The software functions as an overlay for your existing workflow. It creates a local index of your project files. This allows you to search for concepts instead of exact words. When you type a query, the search engine scans your code and documentation to find relevant pieces of information.

The AI browsing feature allows the application to assist with web research. It navigates to pages you request and summarizes content based on your current project goals.

## 🔍 Semantic Search

Traditional search tools look for specific text matches. neuro-nav uses semantic search to understand intent. If you search for "database setup," the software identifies files related to schema definitions and environment configuration, even if the words "database setup" do not appear.

This feature relies on local storage to maintain a fast index. You do not need to send your private code to a remote server for these searches to work.

## 🌐 AI-Powered Browsing

The browser agent helps you gather information from the internet. You provide a link or a topic, and the agent extracts key data. 

- Data extraction: Pulls text or code snippets from documentation pages.
- Contextual summaries: Refines incoming web data to fit your current project tasks.
- Navigation: Automates web page interaction to save time during research sessions.

## 📂 Managing Context

The application keeps your project context tidy. It monitors the files you open and the tasks you perform. It assigns high priority to the files you use most often. This ensures that when you switch between tasks, the assistant knows exactly what you were doing.

Context management prevents you from spending time searching through folder structures. The application remembers where you left off. 

## 🛡️ Privacy and Local Storage

Security remains a focus for this software. The index creation occurs on your machine. Your code remains on your hardware. The application uses your local storage to run its search capabilities. This architecture minimizes data exposure and maximizes performance.

## ❓ Frequently Asked Questions

**Does the software change my existing code?**
No. The application reads your files to create an index. It does not modify, delete, or overwrite your work.

**Can I use this without an internet connection?**
Some features require the internet for AI-assisted tasks. Basic local search functions continue to work while offline.

**How do I update the application?**
The application checks for updates on startup. If a new version exists, it displays a notification with a link to download the updated installer.

**Where does the application store my data?**
All data needed for indexes lives in a hidden folder within your user profile directory. You can clear this cache from the settings menu at any time.
